-------------------------------------------------

#### 讓我們在未來的生活中能在長途旅程中更加的輕鬆。

### **功能性需求**
#### 1. 不需人力駕車
#### 2. 輸入起始地及目的地即可開啟導航模式導引至目的地
#### 3. 可以自行判算各個交通號誌該做甚麼事情
### **非功能性需求**
#### 1. 使用者啟動整個系統需約花30秒鐘
#### 2. 導航系統所導出的路徑未必是最短路徑
#### 3. 使用介面淺顯易懂使用者可以快速上手
